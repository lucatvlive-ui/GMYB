## 🚀 Schnell zum Hochladen - 3 Schritte!

### Schritt 1️⃣: Icons erstellen (5 Min)
```
1. https://realfavicongenerator.net/ öffnen
2. Dein Gym-Logo hochladen
3. Icons runterladen
4. In bilder/ speichern:
   - icon-192.png
   - icon-512.png
   - icon-maskable.png
```

### Schritt 2️⃣: Auf GitHub hochladen (5 Min)
```bash
cd "GMYB (Gym Makes You Better)"
git init
git add .
git commit -m "GMYB PWA - Ready for Deployment"
git remote add origin https://github.com/DEIN_USER/GMYB.git
git push -u origin main
```

### Schritt 3️⃣: Auf Netlify deployen (2 Min)
```
1. https://netlify.com öffnen
2. "New site from Git" klicken
3. GitHub wählen
4. GMYB Repository auswählen
5. "Deploy site" klicken
6. ✅ FERTIG! App live unter xxxxxx.netlify.app
```

**Total Zeit: ~15 Minuten bis Live! 🎉**
